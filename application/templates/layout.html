  • 点赞机是什么

    一个可以帮你自动自动点赞特定的状态/说说的点赞机器人，登陆后还能设定自己的点赞规则，比如每隔多少分钟自动点赞一次。。。
^_^ 或者
while(true) {
  给xxx的说说点赞;
  给xxx的说说取消赞;
  time.sleep(1分钟);//节操呢。。
}

  • 使用点赞机有盗号风险吗

    没有，所有操作跟你在自己的空间操作是一样的，包括登陆

  • 如果不小心点赞了不该点赞的状态怎么办

    你可以过滤调某些词语，使之不会点赞，支持过滤表情, 或者点赞的说说或动态显示出来，你可以立即取消赞
